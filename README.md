## Methodology Principles
 
Across all projects, I follow a consistent set of standards:
 
- **Honest reporting over flattering results.** Project 06 reports a null finding rather than overstating a weak correlation. Project 04 was revised mid-analysis after a data completeness issue was identified.
- **Cross-validation over single-split metrics.** ML models (Project 01, 05) report cross-validated performance, not just best-case single splits.
- **Business framing over technical jargon.** Every project translates model outputs into investment, risk, or due diligence language — Sharpe ratios become client conversations, SHAP values become due diligence narratives.
- **Reproducibility.** All notebooks run end-to-end in Google Colab with documented data sources.
