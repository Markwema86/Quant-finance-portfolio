# Portfolio Analysis Engine

A quantitative analysis tool for evaluating portfolio performance, risk metrics, and asset allocation strategies.

## Contents
- `portfolio_analysis_engine.ipynb` - Main analysis notebook

## Overview
This module provides comprehensive portfolio analysis including:
- Performance metrics (Sharpe ratio, Sortino ratio, Calmar ratio)
- Risk analysis (VaR, CVaR, correlation matrices)
- Asset allocation optimization
- Backtesting capabilities
